# banda.c
banda cachorra- Curriculum Vitae para mis mascotas.
Esta página fue pensada para hablar sobre la gestión de residuos. La excusa para que se parezca a un cv son mis tres perritos cuyos nombres comienzan con "r" (Roco, Romulo y Remo).
Trabaje con resolución de pantalla 1366 x  768 orientación horizontal, en Visual Studio Code

El cv cuenta con tres secciones principales: "que hacemos", "quienes somos" y "trabajos que nos gustaron"; por otro lado un nav que hipotéticamente te permite navegar por dos de estas secciones principales ya que podrían tener mucha más info y contenido e incluso ser hojas de html. A su vez en el navegador hay un botón "tres rrr" que te lleva hacia otra web cuyo contenido tiene que ver con la gestión de residuos.
Hay también un aside con información breve sobre conceptos que abordan los objetivos de lo que hace la banda cachorra.
La sección de "trabajos que nos gustaron" está pensada a modo de galería de fotos, podría ser un carrusel.
La sección de "quienes somos" es una ficha de cada perrito con datos como fecha de nacimiento, juego favorito. Con las propiedades de flexbox dispuse las fichas en forma triangular aludiendo a las tres r (reducir, reciclar, reutilizar)
